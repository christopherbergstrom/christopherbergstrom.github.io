Paragraph

The paragraph is the most used way to get text on the screen.

<p>this is a paragraph tag</p>

Image

Images are used to bring your website to life.

<img src="www.examplesite.com/flower.jpg" alt="flowers">

Link

Links give your website the ability to easily go to other websites.

<a href="https://google.com" target="_blank">Google</a>

Google

List

Lists are a useful way to list information. There are two types of lists, ordered and unordered.

<ol>
<li>Ordered item 1</li>
<li>Ordered item 2</li>
<li>Ordered item 3</li>
</ol>

<ul>
<li>Unordered item 1</li>
<li>Unordered item 2</li>
<li>Unordered item 3</li>
</ul>
  1. Ordered item 1
  2. Ordered item 2
  3. Ordered item 3
  • Unordered item 1
  • Unordered item 2
  • Unordered item 3

Iframe

Iframes are used to embed content from another website into yours. This could be be a whole other website in a little window, or something smaller like a youtube video.

<iframe src="https://responsivedesign.is/articles/xframe-options/" frameborder="0"></iframe>


Form

Forms are used to submit data. However, you can't do much with it unless you know server side code like php or node.js.

<form>
<input type="text" placeholder="username">
<input type="password" placeholder="password">
<input type="submit">
</form>



Input

Inputs are used to collect data. There are many types of input tags such as text, password, submit, radio, checkbox, color, and many more.

<input type="text" placeholder="username">
<input type="password" placeholder="password">
<input type="submit" value="submit">
<input type="radio" name="gender" value="male"> Male
<input type="radio" name="gender" value="female"> Female
<input type="radio" name="gender" value="other"> Other
<input type="checkbox" name="fruit" value="banana"> Banana
<input type="checkbox" name="fruit" value="apple"> Apple
<input type="checkbox" name="fruit" value="orange"> Orange
<input type="color">
