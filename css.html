Selectors

There are two types of selectors to help you target your html elements. The first is id which is specified with "#", and then second is class which is specifies with ".".

Another way to target html elements is to use the element name itself in your css code.

#idName {color: red};
.className {font-weight: bold};
h1 {text-decoration: underline};
